# Protein Fit MTY

Responsive menu and WhatsApp ordering experience built with React and Vite.

## Local development

```bash
npm install
npm run dev
```

The WhatsApp destination is configured in `src/App.jsx` as `WHATSAPP_NUMBER`.

## GitHub Pages

1. Create a GitHub repository and push this project to the `main` branch.
2. Open **Settings > Pages** in the repository.
3. Set **Source** to **GitHub Actions**.
4. The included workflow builds and deploys the site after each push to `main`.

The Vite base path is relative, so the site works on both account-level and project-level Pages URLs.
